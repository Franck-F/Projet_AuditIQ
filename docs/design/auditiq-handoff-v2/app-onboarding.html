Étape 3 / 5 · Cas d'usage prioritaire

Quel système d'IA voulez-vous auditer en premier ?

Choisissez le cas qui vous concerne le plus aujourd'hui. Vous pourrez en ajouter d'autres ensuite. Ce choix oriente le module recommandé.

M1 supervisé Scoring de candidatures (CV / recrutement) Détection de biais par genre, âge, origine. Dataset attendu : 1 ligne / candidature.
M1 supervisé Attribution prime / promotion Audit d'un modèle de décision RH interne.
M2 non supervisé Scoring de crédit / acceptation Détection de proxies (code postal, nom, prénom) sans accès aux attributs sensibles.
M2 non supervisé Tarification assurance Identification de clusters tarifaires déviants.
M3 LLM Chatbot SAV ou support client Test de l'équité des réponses selon le genre, l'origine, le handicap.
M3 LLM Assistant IA interne (RH, juridique) Audit d'un copilote sur 6 axes sensibles.
À noter : Le module recommandé dépend du dataset disponible. Si vous n'avez pas la variable sensible (genre, origine…), AuditIQ vous orientera automatiquement vers le Module 2 (non supervisé) qui cherche des proxies.

Tutoriel guidé — Mon premier audit M1

8 étapes commentées · ~25 min · données d'exemple recrutement.csv

Démarrer le tutoriel →
  1. 01
    Importer le dataset CSV
    412 lignes · 18 colonnes · format détecté automatiquement
  2. 02
    Mapper les colonnes du dataset
    Identifier ID, variables explicatives, décision et attribut sensible
  3. 03
    Choisir la variable cible et l'attribut sensible
    Cible : décision_finale · Attribut : genre
  4. 04
    Configurer les métriques fairness
    4 métriques activées par défaut : DP, EO, EOdds, règle 4/5

    Lancer l'audit (environ 12 min)
    Vous pouvez fermer l'onglet, vous serez notifiée par email
  6. 06
    Lire les résultats — feu tricolore
    Comprendre le score global et le détail par métrique
  7. 07
    Examiner les écarts par groupe
    Voir les graphiques de répartition Hommes / Femmes

    Générer et exporter le rapport
    PDF signé · Excel détaillé · partageable avec votre conseil
