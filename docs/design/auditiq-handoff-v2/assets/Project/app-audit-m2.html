Lecture pédagogique du risque

Sur les 8 240 dossiers de crédit immobilier analysés, le clustering a identifié 5 groupes naturels. Quatre d'entre eux présentent un taux de refus cohérent avec les variables explicatives attendues (revenu, apport, ancienneté). Mais le Cluster #3 (8% des dossiers, soit 660 demandes) présente un taux de refus 2,4× supérieur à la moyenne.

La variable code_postal_2 (premiers chiffres du code postal) explique 61% de la formation de ce cluster. Comme les codes postaux en France sont fortement corrélés à l'origine géographique et socio-économique des résidents, cette variable agit potentiellement comme un proxy d'attributs protégés (origine, situation économique).

Signalement proxy

La variable code_postal_2 domine massivement la formation du cluster.

En France, les premiers chiffres du code postal sont fortement corrélés à :

  • l'origine géographique et migratoire
  • le revenu médian du quartier
  • la composition ethno-culturelle (INSEE 2023)

Cette variable agit donc comme un proxy possible de critères protégés au sens de l'article 225-1 du Code pénal.

Article 225-1 du Code pénal : constitue une discrimination toute distinction opérée en raison de l'origine, du lieu de résidence ou de la situation économique. L'usage d'une variable proxy comme le code postal peut être qualifié de discrimination indirecte, même sans intention discriminatoire. La banque doit pouvoir justifier objectivement que cette variable est un indicateur de risque crédit légitime et qu'il n'existe pas de moyen moins discriminant d'atteindre le même objectif.
